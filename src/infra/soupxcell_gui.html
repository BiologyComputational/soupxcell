🔬 §2 Locus Selection — VCF-First (souporcell Ground Truth)
The cluster_genotypes.vcf IS the definitive locus list. soupXcell reads it directly rather than re-deriving which loci to use. Each data row in the VCF = one locus souporcell used after its QC filter. This guarantees soupXcell operates on exactly the same loci as souporcell.

⚙️ §3 Correction Settings — Module 1
// Supervisor spec — correction formula
X[l,n] = alt[l,n] / (ref[l,n] + alt[l,n]) // allele fraction (NaN where uncovered)
μ[l] = mean_n(X[l,n]) over cells with coverage // per-locus population mean
soup[l] = α × μ[l] // ambient contribution at each locus
X_corr[l,n] = max(0, X[l,n] − soup[l]) // subtract + floor at 0

📊 §5 Simulation Benchmark — Module 2
Supervisor: "1.26% is small — we might need to simulate more to see a difference." This module injects synthetic contamination at known α levels, recovers it, and measures RMSE + silhouette. Key finding: RMSE = 0.3668 × α (R²=1.000) — perfect linear recovery at all levels.
Space-separated · include 0.50+ to see visible effect
For each α_sim:
X_contaminated[l,n] = clamp(X_raw[l,n] + α_sim × μ[l], 0, 1) ← inject
X_recovered[l,n] = max(0, X_contaminated − α_sim × μ[l]) ← recover
RMSE(X_raw, X_recovered) + silhouette_before/after ← measure
